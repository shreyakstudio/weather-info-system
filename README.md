# Weather Information System

## Problem Definition

Modern users need quick access to real-time weather data without navigating 
complex interfaces. This project solves the problem of providing a clean, 
accessible weather dashboard that displays:
- 🌡️ Temperature (°C)
- 💧 Humidity (%)
- 💨 Wind Speed (km/h)
- 🤔 Feels Like (°C)

## Objectives
- Build a lightweight weather web application using Node.js
- Integrate with OpenWeatherMap API for real-time data
- Implement a complete CI/CD pipeline using GitHub Actions
- Containerize using Docker and deploy to a staging environment

## Relevance
Addresses real-world need for accessible weather information with a 
fully automated DevOps workflow demonstrating CI/CD best practices.
